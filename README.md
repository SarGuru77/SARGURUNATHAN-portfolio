A modern, highly interactive, and responsive developer portfolio website built from scratch using clean vanilla technologies. Designed to showcase my personal profile, skills, services, and featured projects.

---

## ✨ Features

- **Responsive Grid Layout**: Fully optimized and pixel-perfect for mobile, tablet, and desktop screens.
- **Dynamic Typewriter Animation**: Showcases multiple professional titles dynamically on the hero section.
- **Interactive Particle Background**: Interactive node-connecting canvas particles that respond to viewport changes.
- **Cursor Glow Light**: Subtle cursor-tracking overlay glow that follows mouse movements on desktops.
- **Dynamic Skills Tabs**: Interactive tab switching to organize Backend, Frontend, Databases, and DevOps skills.
- **Modern Dark Aesthetics**: Premium HSL-based dark mode colors, neon accents, and smooth hover micro-animations.
- **Working Contact Form**: Clean form styling with floating label transitions.

---

## 🛠️ Built With

No complex frameworks, bundlers, or heavy libraries needed:
* **HTML5**: Semantic tags structured for SEO and accessibility.
* **CSS3**: Modern layouts using Flexbox and Grid, CSS Custom Variables for theme management, and keyframe animations.
* **JavaScript (ES6+)**: Custom dynamic particle systems, typing animations, scroll-triggered navigation highlight, and stats counter animation.
